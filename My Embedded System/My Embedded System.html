My Embedded System; Pressure Activated LED

What this embedded system does.

The main thing that this system completes is that when you press down the disk on the Analog Piezo Disk Vibration Sensor, once you go over a certain threshold the LED will turn on due to how much pressure is place on a disk.

Equipment

  • Computer with the Arduino IDE software on it
  • Arduino Uno
  • USB 2.0 Cable Type A/B
  • Jumper Wires
  • Connection Cable for Analogue Sensors
  • A Analog Piezo Disk Vibration Sensor
  • A LED
  • A Resistor
